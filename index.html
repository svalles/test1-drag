Explicación de Paginación

La paginación en x86 usa una tabla de páginas jerárquica:

  1. Registro CR3 apunta al directorio de páginas
  2. Directorio de páginas (PDE)
  3. Tabla de páginas (PTE)
  4. Marco de página física

Traducción de Direcciones

Dirección virtual (32 bits):

[10 bits PDE] [10 bits PTE] [12 bits Offset]
CR3: 0x00000000
Page Directory
PDE 0: 0x00001000
Page Table
PTE 0: 0x00002000
